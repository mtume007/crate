Philosophy

HULDRA is a personal design system named after the Norwegian forest spirit — beautiful on the surface, precise and modular underneath. It is the design language for a suite of applications built around one belief: that software for people who care about their collections should feel like the collection itself. Considered. Calm. Timeless.

The UI is the room the collection lives in.
The room doesn't perform.

Three references hold the system together. Wegner and Jacobsen — honest materials, structure visible but never decorative. James Turrell — the interface tries to disappear. You are inside your music, not looking at it. Apple Liquid Glass — controls float above content, recede when not needed.

Design principles

01

One font

Martian Mono throughout. Weight and size carry all hierarchy. The whole app speaks one typographic language.

02

Amber is structural

The accent appears only on labels — YEAR, GENRE, BPM. Never on content. Artist names are white. "2002" is muted. Absolute.

03

Nothing hardcoded

Every value lives in config. Every colour is a token. Swap the accent, toggle the mode — the whole system follows.

04

Spring motion

Cards scale with a slight overshoot on hover. The UI is alive at the point of contact. Only transform and opacity — never layout.

05

Two modes

Dark is near-black #080808. Light is warm parchment #f5f0e8. The amber accent shifts to match. Same personality, different room.

06

Glass layers

Toolbar, titlebar, and playbar float above content with backdrop blur. The chrome recedes. The music comes forward.

Crate — the first HULDRA application

Crate is a local-first desktop music library for audiophiles, DJs, and collectors. Built for people who have spent years accumulating records, digital files, and rare pressings — and want a single place that treats that collection with the seriousness it deserves.

Not a streaming interface. Not a playlist manager. A library and a database — the kind of thing a serious collector would build for themselves if they could.

Scans local music folders, reads file tags, deduplicates intelligently
Enriches every album via Discogs — label, cat#, pressing, year, format
AI-assisted matching — Claude normalises queries, validates results
Review pipeline — auto-flags low-confidence matches for quick confirm or manual correction
Detail panel shows cat#, label, runtime alongside artwork
Single-album add flow — drop a folder, pick the pressing, done
Dark and light mode, fully themeable from config
409 albums · 4 564 tracks — and growing
